Boot Logo for OS/2 Warp Version 4
---------------------------------

ATTENTION! This boot logo is only to be
installed on OS/2 Warp 4. With older versions
of OS/2, this logo will cause trouble.

Usual reminder:
-------------------
Installing this can crash your software
or hardware. Use at own risk. 
NO WARRANTY!

Installation:
-------------
1. So you already unzipped the zip file ;-)

2. Open OS/2 window and type as follows:

   [C:\]attrib -s -r -h os2logo
   [C:\]ren os2logo os2logo.bak
   [C:\]exit

3. Now copy the "os2logo" file on your 
   OS/2 boot drive, root directory.

4. At next reboot, you will see the new
   boot logo.

Deinstallation
--------------
1. Open OS/2 window

2. [C:\]ren os2logo os2logo.weg
   [C:\]ren os2logo.bak os2logo
   [C:\]exit

3. At next reboot, you will see the old
   boot logo.
